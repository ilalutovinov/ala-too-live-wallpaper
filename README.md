# Bishkek Live Cam - Ala-Too Square 🇰🇬

Живые обои для **Wallpaper Engine** с прямой трансляцией с главной площади Бишкека — Ала-Тоо.

## ✨ Особенности

*   **Real-time поток:** Прямая трансляция в режиме реального времени.
*   **Авто-восстановление:** Обои оснащены умной системой автоматического обновления потока (каждые 10 минут). Это гарантирует стабильную работу без задержек, лагов и переполнения оперативной памяти.
*   **Оптимизация:** Минимальное потребление ресурсов системы и плавное переключение при выходе компьютера из спящего режима.

---

## 🚀 Как использовать в Wallpaper Engine

1.  Скопируйте ссылку на ваш GitHub Pages сайт (вид: `https://your-username.github.io/wallpaper-cam/`).
2.  В **Wallpaper Engine** нажмите кнопку **«Открыть URL»**.
3.  Вставьте скопированную ссылку в поле URL.
4.  **ВАЖНО:** Обязательно поставьте галочку **«Использовать режим трансляции»**.
5.  Нажмите «ОК» и наслаждайтесь видом на Ала-Тоо.
